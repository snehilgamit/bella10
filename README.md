## Getting Started

### Prerequisites

- Node.js (>=14)
- npm / yarn
- MongoDB instance (or any database configured)

### Installation

```bash

git clone https://github.com/snehilgamit/bella10.git

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration
